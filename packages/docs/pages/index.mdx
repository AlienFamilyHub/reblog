# reblog

`reblog`是一个基于`go`编写的，轻量的自托管博客系统。

## 特性

- 轻量化：基于`go`语言编写，编译后二进制文件体积`~20MB`。
- 容器化：提供`docker`和`docker-compose`支持。
- 高自由度：提供`ThemeKit`用以主题获取数据，只提供数据，不限制框架。
